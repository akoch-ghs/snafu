SNAFU SNow Automated Form Utilizer


   

What is SNAFU?

SNAFU stands for Service Now Automated Form Utilizer. This is an extension for Google Chrome that is designed to be a replacement to the Service Now Automation Program (SNAP) we used as Dell/NTT DATA employees to streamline some of the mundane tasks that needed to be done to properly track our tickets. After Desktop Services were brought in-house by GHS we lost access to SNAP since it was developed by Dell. I spoke our manager, and got his approval, so I did some research in to the feasability of a Chrome Extension as a replacement. Once I discovered it was technically possible I started learning how to write extensions, as well as Service Now's powerful API to interact with the page. SNAFU is the result.
