SNAFU Options


Options
Ticket Completion
Action to take when you submit an update.

Completion Delay
Delay, in seconds, before executing the action above.

Close Popup
When sending an update do you want the popup window to close?

Persistent Notes
Persist notes between popup closures until update it sent.

Send On Enter
Pressing [ENTER] when typing a note will send the update. Hit Ctrl/Shift/Alt+ENTER for new a new line.

Close Alerts
Automatically close the Epic Hyperspace and BCA device alerts that appear in incidents.

Put each message on its own line in the format of {label}|{message}.

Debug Mode
You can leave this disabled unless told to enable.
